# 🎮 学習継続アプリ（Study Coin App）

Streamlit を利用した、学習の継続をゲーム感覚でサポートする Web アプリケーションです。  
学習内容と学習時間を記録することでコインを獲得でき、**コインやレベルに応じて応援や褒めるメッセージを表示**する機能を搭載したので、モチベーションを維持しながら学習を続けることができます。

また、**Supabase を用いたデータの永続化**により、  
ブラウザを閉じても学習記録が保持される仕組みになっています。  
さらに、**外部 Web API を利用したデータ取得機能**や、  
**エラー発生時にもアプリが停止しないようにする例外処理（try-except）**を実装し、安定して動作するアプリを目指しました。

## 🌟 主な機能

- 📘 **学習記録機能**  
  学習内容と学習時間（分）を入力し、1日に何回でも学習を記録できます。

- 💰 **コイン獲得システム**  
  学習時間10分につき1コインを獲得します。

- ⭐ **レベル表示機能**  
  累計コイン数に応じてレベルが自動的に計算・更新されます。

- 🗒️ **学習履歴の表示**  
  当日の学習履歴を時刻・内容・獲得コインとともに一覧表示します。

- ☁ **データ永続化（Supabase）**  
  Supabase を利用して学習データを保存しているため、  
  ブラウザを閉じても学習状況が保持されます。

- 🌐 **外部 Web API 連携機能**  
  外部の Web API からデータを取得し、アプリ内で活用しています。

- 🛡️ **エラーハンドリング（try-except）**  
  API 通信エラーやデータ取得失敗時にも、  
  アプリがクラッシュせずユーザーに分かりやすいメッセージを表示します。

## 📊 データの仕組み

このアプリでは **Supabase（PostgreSQL）** を用いて学習データを管理しています。

### study_logs テーブル

| カラム名 | 内容 |
|--------|------|
| id | ID |
| study_date | 学習日 |
| study_time | 学習時刻 |
| topic | 学習内容 |
| minutes | 学習時間（分） |
| coins | 獲得コイン数 |
| created_at | 記録作成日時 |

### データ処理の流れ

1. 学習完了時に学習ログを Supabase に保存  
2. アプリ起動時に Supabase から学習ログを取得  
3. 外部 Web API から必要なデータを取得  
4. エラー発生時は try-except により安全に処理  
5. 保存されたデータからコイン総数を計算  
6. コイン数に応じてレベルを自動算出
