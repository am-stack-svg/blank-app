# 🎮 学習継続アプリ（Study Coin App）

Streamlit を利用した、学習の継続をゲーム感覚でサポートする Web アプリケーションです。  
学習内容と学習時間を記録することでコインを獲得でき、モチベーションを維持しながら学習を続けることができます。

## 🌟 主な機能

- 📘 **学習記録機能**  
  学習内容と学習時間（分）を入力し、1日に何度でも記録できます。

- 💰 **コイン獲得システム**  
  学習時間10分ごとに1コインを獲得できます。

- ⭐ **レベル表示**  
  累計コイン数に応じてレベルが自動的に計算・表示されます。

- 🗒️ **学習履歴の表示**  
  当日の学習履歴を一覧で確認できます。

- ☁ **データ永続化**  
  Supabase を利用して学習データを保存しているため、  
  ブラウザを閉じてもコインやレベルが保持されます。
